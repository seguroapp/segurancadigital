# Seguranca na Internet e LGPD

Landing page estatica em HTML, CSS e JavaScript com foco em educacao digital, privacidade e seguranca cibernetica para o uso cotidiano.

## O que foi atualizado

- Layout responsivo com visual mais moderno e corporativo
- Navegacao unica por secoes, substituindo links quebrados
- Checklist interativo de seguranca com progresso
- Diagnostico rapido de risco digital
- FAQ expansivel
- Formulario de contato com validacao local

## Estrutura

- `index.html`: conteudo principal da pagina
- `style.css`: identidade visual e responsividade
- `script.js`: interacoes da interface
- `logotipo.mp4`: elemento visual da marca

## Como abrir

Basta abrir `index.html` no navegador.

## Publicar no Netlify

1. Crie uma conta gratuita no Netlify.
2. No painel, use `Add new site` > `Deploy manually`.
3. Arraste a pasta inteira do projeto para o upload.
4. O site sera publicado com HTTPS em um dominio temporario do Netlify.

### Formulario de contato

O formulario ja esta preparado para `Netlify Forms`.

- Localmente, ele apenas valida e simula o envio.
- No Netlify, as mensagens passam a ser capturadas no painel do site.

### Observacao

Como o projeto e estatico, nao existe etapa de build obrigatoria.
